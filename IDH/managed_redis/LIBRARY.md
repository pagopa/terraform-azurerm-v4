## Configuration Reference

Each tier configuration should include:

```yaml
your_tier_name:
  # SKU and sizing
  sku_name: "Enterprise_E5"  # or Enterprise_E10, E20, E50, E100, EnterpriseFlash_F300, F700, F1500
  
  # High Availability
  high_availability_enabled: false
  
  # Network
  public_network_access: "Disabled"
  private_endpoint_enabled: true
  
  # Database configuration
  access_keys_authentication_enabled: true
  client_protocol: "RESP3"
  clustering_policy: "EnterpriseCluster"
  eviction_policy: "allkeys-lru"
  
  # Persistence (optional)
  persistence_configuration:
    rdb_enabled: false
    aof_enabled: false
  
  # Modules (optional)
  modules: []
  # modules:
  #   - name: "RediSearch"
  #   - name: "RedisJSON"
  #   - name: "RedisBloom"
  
  # Monitoring and alerts
  enable_cpu_alerts: false
  cpu_usage_percentage_threshold: 80
  enable_memory_alerts: false
  memory_usage_percentage_threshold: 80
  enable_eviction_alerts: false
  enable_connection_alerts: false
  connection_count_threshold: 5000
```
